# Frameworks Using

Frontend: ReactJS with TypeScript  
Backend: Spring Boot  
Database: MySQL

# Deploy JAR
```bash
$ cd server
$ ./mvnw clean install -s .mvn/wrapper/settings.xml -DskipTests
```

# How to Compile With Docker

```bash
$ docker-compose up -d --build
```
